✅ AI trong app App chấm ngay trên giao diện này.
Viết câu rồi bấm “AI chấm câu”. Kết quả sẽ hiện ở đây.
Nâng cao: kết nối ChatGPT thật qua server nội bộ
Key ChatGPT không nhập trong HTML. Nếu dùng ChatGPT thật, key nằm trong file .env của server kèm theo, nên học viên chỉ bấm “AI chấm câu”.
Nếu server chưa chạy, app vẫn chấm bằng AI Albert nội bộ 3.0. Bộ chấm mới kiểm tra ngữ cảnh, mẫu câu, loại từ, collocation, logic câu và gợi ý sửa cụ thể hơn.
